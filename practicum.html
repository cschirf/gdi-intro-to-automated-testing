Without Doubles

Problem Statement

Return the sum of two 6-sided dice rolls, each in the range 1..6. However, if noDoubles is true, if the two dice show the same value, increment one die to the next value, wrapping around to 1 if its value was 6.

Each die value is an integer. The flag noDoubles is a boolean.

Function header:
withoutDoubles(die1, die2, noDoubles)

(Original problem withoutDoubles from codingbat.com)

Without Doubles

Example implementation


var withoutDoubles = function(die1, die2, noDoubles) {
  if (noDoubles && die1 === die2) {
    if (die1 === 6) {
      return die1 + 1;
    }
    return die1 + die2 + 1;
  }

  return die1 + die2;
};

Introduction to Test-Driven Development

TDD Approach: Review

  • Step 1: Red
    • Write test cases. The tests should fail for the expected reason(s) (e.g., because the implementation is not correct, not compile or runtime errors). At this point, check failure messages to ensure they accurately indicate the problem. Fix wording as necessary.
    • Write shell— write enough code to get the tests to compile
  • Step 2: Green
    • Write just enough code to get the tests to pass. For a complex problem complex, you might consider taking an iterative approach to the code— e.g. write enough code to get part of the test suite to pass, then work on the next piece.
  • Step 3: Refactor
    • Take another pass over the code and refactor as necessary. Re-run the tests to ensure the refactor has not broken functionality.

Observations

  • We only want to write just enough code to get the tests to pass. If we can remove a piece of code, but the tests still pass, then that code is not doing anything but bloating up the codebase. What happens if we remove !isWeekend in the second check?
  • It would be nice if we pulled out 40 and 60 into named constants, instead of leaving them as magic numbers in the codebase. Helper Rounding Function


function roundBetter(number, precision) {
    var factor = Math.pow(10, precision);
    var tempNumber = number * factor;
    var roundedTempNumber = Math.round(tempNumber);
    return roundedTempNumber / factor;
}

Example refactored implementation 


  var DIVISOR = 1000;
  var NUM_DECIMAL_PLACES = 1;
  var MAX_RECURSIONS_ALLOWED = 4;

  var MAGNITUDE_TO_LETTER_ABBREVIATION = {
    0: '',
    1: 'K',
    2: 'M',
    3: 'B',
    4: 'T'
  };

  function roundBetter(number, precision) { ... }

  function formatNumberStr(input, numRecursions) {
    var numAsStr = roundBetter(input, NUM_DECIMAL_PLACES).toFixed(NUM_DECIMAL_PLACES).toString();
    return numAsStr + MAGNITUDE_TO_LETTER_ABBREVIATION[numRecursions]
  }

  function recursivelyFormatNumber(input, numRecursions) {
    if (Math.abs(input) < DIVISOR || numRecursions === MAX_RECURSIONS_ALLOWED) {
      return formatNumberStr(input, numRecursions);
    }

    return recursivelyFormatNumber(input/DIVISOR, numRecursions+1);
  }

  function formatNumber(input) {
    if (Math.abs(input) < DIVISOR) {
      return input.toString();
    }

    return recursivelyFormatNumber(input, 0);
  }
